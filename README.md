**Deskripsi**

1. Judul : Loan Approval

2. Dataset URL : https://www.kaggle.com/datasets/rohit265/loan-approval-dataset

**Metode**

1. Logistic Regression → metode/algoritma klasifikasi utama

2. SMOTE → teknik preprocessing untuk menangani ketidakseimbangan kelas

3. One-Hot Encoding → teknik transformasi fitur kategorikal

4. Train–Test Split (70:30, stratified) → strategi pembagian data

5. Evaluasi → Accuracy, Precision, Recall, F1-Score, AUC, Confusion Matrix
